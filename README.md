# Discounted Cash Flow (DCF) Valuation Model

## Overview
Built a DCF valuation model in Excel to estimate the intrinsic value of a publicly traded company.

## Key Features
- 3-Statement Financial Model
- 5-Year Revenue Forecast
- Unlevered Free Cash Flow Projection
- WACC Calculation using CAPM
- Terminal Value Estimation
- Sensitivity Analysis

## Skills Demonstrated
- Financial Modeling
- Valuation
- Corporate Finance
- Excel
- Business Analysis

## Results
- WACC: 6.24%
- Implied Share Price: $358.48
- Sensitivity Range: $231 - $895
